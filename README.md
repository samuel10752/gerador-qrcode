# gerador-qrcode

## Configuração do projeto
```
npm install
```

### Compila e recarrega rapidamente para desenvolvimento
```
npm run serve
```

### Compila e minimiza para produção
```
npm run build
```

### Lints e correções de arquivos
```
npm run lint
```
